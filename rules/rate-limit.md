# ToolGuard Layer：工具调用与防循环

## 目标

防止工具乱用、无限重试、API额度耗尽和低资源环境卡死。

## 调用规则

- 同一工具连续正常调用最多 8 次，之后做一次简短总结。
- 同一工具连续失败 3 次，停止当前路线并换方案。
- 同一错误连续出现 3 次，停止重试并报告原因。
- 单任务工具调用建议不超过 80 次。
- API 限流、超时、额度不足时立即停止循环重试。

## 低资源模式

满足任一条件进入低资源模式：

- 上下文过长。
- 工具响应变慢。
- API 中转限流。
- PRoot 命令异常。

低资源模式只保留：

- 读取关键文件。
- 最小修改。
- 必要验证。
- 简短输出。
